Встроенные модули — это модули, которые уже входят в состав Python и не требуют установки через pip.
Их можно использовать сразу, просто импортировав в код с помощью команды import.

Примеры встроенных модулей Python:

1. os — используется для работы с файлами и папками (создание, удаление, переименование).
Пример: os.listdir() показывает список файлов в папке.
2. sys — помогает взаимодействовать с системой и параметрами запуска программы.
Пример: sys.argv хранит аргументы, переданные при запуске скрипта.
3. datetime — нужен для работы с датами и временем.
Пример: datetime.now() возвращает текущие дату и время.
4. json — используется для чтения и записи данных в формате JSON (часто в API или настройках).
Пример: json.dump() сохраняет словарь Python в файл.
5. random — генерирует случайные числа.
Пример: random.randint(1, 10) возвращает случайное число от 1 до 10.
